# simple-startscsser
Hopes to get you started for TypeScript and SCSS. Tests? What's that?
- Reload when HTML modified.
- Reload when SCSS modified.
- Reload when TS modified.

### DEV
``npm run start``

### DIST
``npm run build``

### USE
- Download or clone.
- Run ``npm install``
- Run DEV or DIST.
